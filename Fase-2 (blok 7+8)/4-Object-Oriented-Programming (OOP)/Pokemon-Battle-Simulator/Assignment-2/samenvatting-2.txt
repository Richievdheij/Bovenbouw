# The assignment contains the following concepts: composition, reference types, and value types.
# The assignment uses the following keywords: null, List<Pokeball> (with C#) or ArrayList<Pokeball> (with Java).

# Composition:

Composition is het concept waarbij we objecten combineren om complexere structuren te maken, zoals objecten binnen objecten.
Dit helpt bij het modelleren van real-world scenario's. 

# Reference Types:

Reference types zijn datatypes die verwijzen naar geheugenlocaties waar de werkelijke waarden zijn opgeslagen. 
Ze hebben een toegewezen geheugenruimte om de verwijzing naar de waarden op te slaan.

# Value types:

Value types slaan daarentegen waarden direct op in de variabele zelf,
zonder verwijzingen naar geheugenlocaties.

# Null:

"Null" wordt gebruikt om aan te geven dat een referentietype geen verwijzing naar een object bevat.

# List:
Een "List<Pokeball>" is een datatype in C# dat een geordende verzameling van "Pokeball" objecten opslaat,
waardoor we flexibiliteit hebben bij het werken met verschillende objecten.
